# Fake News Detection using NLP and LSTM

## 📌 Project Description
This project detects whether a news article is real or fake using Natural Language Processing (NLP) and an LSTM deep learning model.

## 🛠️ Technologies Used
- Python
- TensorFlow
- Keras
- NLTK
- Pandas
- NumPy
- Scikit-learn

## 📂 Project File
- newsdetection.ipynb

## 🎯 Features
- Text preprocessing
- Tokenization
- Padding
- LSTM model
- Fake/Real news prediction
